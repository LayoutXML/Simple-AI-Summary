# Simple AI Summary

A lightweight, private browser extension that lets you send selected text directly to your favorite AI provider.

## Features
- **100% Private**: Your data never touches our servers because there are no servers.
- **No Middleman**: Direct communication between your browser and the AI provider.
- **No Tracking**: No analytics, no cookies, no data collection.
- **Support**: ChatGPT, Gemini, Perplexity, Claude, or any custom URL.
- **Open Source**: Verify the code yourself.

## Permissions

- **Context Menu**: Adds the "Summarize" option to your right-click menu for quick access.
- **Storage**: Securely saves your preferences (favorite AI provider, custom prompts) locally in your browser.
- **Scripting & Host Permissions**: Allows the extension to automatically paste the selected text into the AI provider's chat box.
- **Active Tab**: Used to capture the text you have selected on the current page.
